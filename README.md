# Alwin Jaison — Software Developer Portfolio

A modern, high-performance personal portfolio built to showcase my projects, skills, and experience as a full-stack developer.

## Tech Stack
- Vite
- React (JSX)
- Custom Vanilla CSS
- Lucide React Icons

## How to Run Locally

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
